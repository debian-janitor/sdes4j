sdes4j
======
Implementation of RFC4568: Session Description Protocol (SDP) Security Descriptions for Media Streams for Java.

Build
-----
Run `mvn package`

Projects using sdes4j
---------------------
[Jitsi](https://jitsi.org), the open source VoIP and Instant Messaging Client.
